Tech Stack
Backend: Node.js, Express
Database: sql
Testing: Jest
## Prerequisites

- Node.js (>=14.x)
- npm (>=6.x)

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/Teerth67/ZEOTAP.git
    ```

2. Navigate to the project directory:
    ```sh
    cd ZEOTAP
    ```

3. Install dependencies:
    ```sh
    npm run install-dependencies
    ```

## Scripts

- **Start the application:**
    ```sh
    npm start
    ```

- **Run tests:**
    ```sh
    npm test
    ```
